적용 방법
1) 기존 프로젝트 폴더의 main.py를 이 파일로 교체합니다.
2) org_url_resolver.py는 속도 개선 로직이 포함된 버전입니다. 이미 동일 버전을 쓰고 있으면 교체하지 않아도 됩니다.
3) 실행: streamlit run main.py

변경 내용
- 후보 검색 페이지, 점수 기준, 브라우저 검색, URL 수집 페이지, 최대 URL 수 입력 UI 제거
- 내부 고정값으로 처리: 후보 검색 3페이지, URL 수집 5페이지, 점수 기준 80, 상세 fallback 5건
- 후보 목록/선택박스에는 제공기관명만 표시
- 선택 후 확인 버튼 클릭 시 URL 수집까지 바로 진행하고 메타데이터 크롤링 메뉴로 이동
- URL 샘플/점수/예시 URL 표시 제거
- 선택박스 글자는 흰색, 나머지 밝은 박스 내부 글자는 검정색 기준으로 정리
