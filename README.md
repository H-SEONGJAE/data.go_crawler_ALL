# 공공데이터포털 통합 크롤러 - URL Resolver 개선/브라우저 오류 대응 버전

## 실행

```bash
pip install -r requirements.txt
streamlit run main.py
```

파일 다운로드 기능처럼 브라우저가 반드시 필요한 기능을 사용할 경우 아래 명령을 먼저 실행하세요.

```bash
python -m playwright install chromium
```

기관 후보 검색/URL 검증은 기본적으로 **requests 기반**으로 동작하므로 Playwright 브라우저가 없어도 먼저 실행할 수 있습니다. 화면에서 `브라우저 기반 검색` 옵션을 켠 경우에는 Playwright가 필요합니다.

## 이번 수정 사항

- 기관 URL을 `org=기관명`으로 직접 조립하지 않음
- 기본 URL 검색은 Playwright 없이 requests로 수행
- 상세페이지의 `제공기관` 값을 다시 읽어 기관 후보 생성
- 선택 기관과 상세페이지 제공기관명을 교차 검증한 URL만 수집 대상으로 확정
- Playwright 브라우저 누락 시 자동 설치 시도 및 명확한 안내 메시지 출력
- UI를 밝은 톤의 흰색 박스/검정 글씨 중심으로 재구성
- 기능별 메뉴는 왼쪽 사이드바에 배치

## 권장 사용 순서

1. `기관 검색 · URL 검증` 메뉴에서 검색어 입력
2. `브라우저 기반 검색`은 끈 상태로 후보 검색
3. 후보 기관 선택
4. URL 교차 검증
5. 검증 통과 URL 개수 확인
6. 메타데이터/조회수/파일다운로드 기능 실행

## 주의

파일 다운로드 기능은 브라우저 다운로드가 필요하므로 Playwright Chromium 설치가 필요합니다.
